## App

You can run the app by using

```r
install.packages(c("dplyr", "shiny", "shinymaterial", "plotly", "viridisLite", "tidyverse", "shinyWidgets", "DT", "RColorBrewer", "shinyjs", "leaflet", "leaflet.extras", "scales"))

shiny::runApp("covid19-dashboard")
```
